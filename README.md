# reMarkable.Tarot.Journal.Template
a reMarkable2 compatible template for tarot card readings

I was frustrated by the lack of good one-page tarot spread journal templates available for the reMarkable2 that I was gifted, so I used [NoTeTo](https://noteto.needleinthehay.de/) to lay out a basic design, and then added moon phase icons using [GraphicConverter](https://www.lemkesoft.de/en/products/graphicconverter).

Including the exported JSON (for reimporting to NoTeTo) and the final PNG.

Hope someone else finds this useful.
